# Command Reference

Install globally (recommended): `pipx install onelap-strava-sync`

- Default sync: `onelap-sync`
- Sync since date: `onelap-sync --since 2026-03-01`
- Download only: `onelap-sync --download-only --since 2026-03-01`
- Strava OAuth init: `onelap-sync --strava-auth-init`

Development fallback from repo root: `python run_sync.py`
